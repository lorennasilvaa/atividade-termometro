# Atividade Termômetro
Repositório contendo exercícios resolvidos na linguagem C para a disciplina Linguagem de Programação I. 

## Estrutura do Repositório

O repositório apresenta dois diretórios: 
- `exercises/`: contém as soluções dos exercícios;
- `docs/`: contém a descrição das questões.  

## Como compilar  

### Exemplo usando gcc:  

`gcc exercises/exercicio01.c -o exercicio01`  

Executar:  

`./exercicio01`  

## Requisitos 

- gcc
